1 📌 Introdução

O experimento da catapulta foi realizado com o objetivo de analisar a variabilidade das distâncias de lançamento de uma bolinha, considerando diferentes fontes de variação.

Foram realizadas 40 observações, permitindo a aplicação de conceitos de estatística descritiva e análise gráfica.

2 🎯 Objetivos

2.1 Objetivo geral

Analisar estatisticamente os dados obtidos no experimento da catapulta.

2.2 Objetivos específicos

  • Calcular medidas de posição e dispersão
  • Construir gráficos
  • Analisar distribuição dos dados
  • Avaliar assimetria e curtose
  • Comparar dados agrupados e não agrupados

4 Resultados e Discussões

  • Os resultados mostram que os dados apresentam variação significativa entre os lançamentos.

  • O histograma permite observar a distribuição dos dados, enquanto o boxplot evidencia possíveis outliers.

  • A análise de assimetria indica se os dados são inclinados à direita ou esquerda, enquanto a curtose mostra o grau de concentração dos dados.

  • A comparação entre dados agrupados e não agrupados evidencia perda de precisão quando os dados são agrupados.

5 Possíveis erros experimentais

  • Variação na força aplicada
  • Diferença na posição da catapulta
  • Influência do ambiente (vento, superfície)

6 Conclusão

  • O experimento permitiu compreender a variabilidade dos dados e a importância das ferramentas estatísticas.

  • As medidas descritivas e os gráficos foram fundamentais para interpretação dos resultados.

## 📊 Dados coletados

```{r}
distancias <- c(
```

## Resultados e Discussões

- Os resultados mostram que os dados apresentam variação significativa entre os lançamentos.

- O histograma permite observar a distribuição dos dados, enquanto o boxplot evidencia possíveis outliers.

- A análise de assimetria indica se os dados são inclinados à direita ou esquerda, enquanto a curtose mostra o grau de concentração dos dados.

- A comparação entre dados agrupados e não agrupados evidencia perda de precisão quando os dados são agrupados.

## Possíveis erros experimentais

- Variação na força aplicada
- Diferença na posição da catapulta
- Influência do ambiente (vento, superfície)

## Conclusão

- O experimento permitiu compreender a variabilidade dos dados e a importância das ferramentas estatísticas.

- As medidas descritivas e os gráficos foram fundamentais para interpretação dos resultados.
